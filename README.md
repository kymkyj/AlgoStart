### 알고리즘 스터디
### 시간 : 매주토요일 오전 10~12시

- 깃에 개인폴더를 만들어서 그안에 표기
- 매주 정해진 문제를 풀어서 PR로 날려놓고 모임에 참여한다.
- 깃 커밋규칙
  1. 새로 파일을 올리는 경우 commit message에 [feat] 내용 형태로 올린다.
  2. 기존에 올렸던 파일을 수정해서 올려야하는 경우 [fix] 내용 형태로 올린다.<br>
  ex) [feat] 스킬트리 코드 커밋
